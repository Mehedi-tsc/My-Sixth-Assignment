## Project Name: 

**Digital Tools Marketplace**

## Description:

This is a responsive website where people can buy digital tools. It has a shopping cart where the tools that the customer clicks to buy will be stored. It is easy to use.

## Technologies Used:

- React.js
- Tailwind CSS
- DaisyUI
- React Toastify
- JSON
- Google Font

## Features
- Products and cart toggle
- Dynamic cart system
- Toast Notification
